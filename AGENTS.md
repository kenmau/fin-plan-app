# FinPlan - Household Financial Planner

## Project
Nx monorepo: React SPA (Vite) + Fastify API + shared TypeScript types package.
Stack: React 18, Fastify 4, Drizzle ORM, PostgreSQL 16, TanStack Query 5, Zustand, Tailwind, shadcn/ui, Recharts.
